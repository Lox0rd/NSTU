# Рекурсивый алгоритм для поиска в диапазоне числа, делящегося на числа от 2 до 18

**Вариант задания:** 9

## Что делает программа

Программа:
* рекурсивно находит НОК для чисел;
* перебирает все возможные комбинации чисел состоящих из цифр;
* находит числа, делящиеся на НОК;
* выводит все числа, подходящие условию.

## Как использовать
**1. Скомпилируйте программу:**

```bash
gcc -o rek_ch main.c
```

**2. Запустите программу**
```bash
./rek_ch
```

**Пример**
Входные данные:
```bash
Все числа состоящие из цифр 0-9
делятся на 2-18
```
Результат: 
```bash
Поиск числа из цифр 0-9, делящегося на все числа от 2 до 18...
Найденый НОК: 12252240
Найдено число: 4876391520
Результат деления на НОК: 4876391520 % 12252240 = 0
```
