# Сортировка слов из файла

**Вариант задания:** 9

## Что делает программа

Программа:
* читает слова из текстового файла;
* сортирует их по алфавиту;
* сохраняет в двухуровневый динамический массив (`char ***`);
* при переполнении массива создаёт новый и переносит туда половину элементов.

## Как использовать
**1. Скомпилируйте программу:**

```bash
gcc -o word_sorter main.c
```

**2. Запустите программу:**
```bash
./word_sorter
```

**Пример**

Содержимое input.txt:
```bash
яблоко банан апельсин груша яблоко банан
```
Результат: слова отсортированы по алфавиту и размещены в массивах заданной размерности.
