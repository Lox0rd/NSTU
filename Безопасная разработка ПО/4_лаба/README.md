# Краткий отчёт по проекту

## 📌 Описание

Проект реализует основные структуры данных:
- стек
- очередь
- бинарное дерево

Реализация выполнена на двух языках:
- C++
- Python

---

## 🧩 C++ часть

Используются:
- модульные тесты (Google Test)
- анализ покрытия кода (gcov + lcov)
- генерация HTML отчёта покрытия

Структура включает отдельные модули для каждой структуры данных.

---

## 🐍 Python часть

Реализована аналогичная структура данных(stack).

Используются:
- `unittest` для тестирования
- `coverage.py` для анализа покрытия
- виртуальное окружение `venv`

---

## 📊 Результат

- реализованы базовые структуры данных
- написаны тесты для проверки корректности
- выполнен анализ покрытия кода в обеих реализациях
- сгенерированы HTML отчёты покрытия

---

## ⚙️ Инструменты

- C++: g++, Google Test, lcov, gcov
- Python: venv, unittest, coverage.py

Перед запуском проекта обязательно прочитайте файлы `README.md` в других папках — там содержится подробная инструкция по настройке и запуску.
