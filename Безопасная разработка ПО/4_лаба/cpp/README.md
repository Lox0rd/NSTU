# Cpp часть проекта

Проект на C++, реализующий:
- стек
- очередь
- бинарное дерево

В проекте используются:
- модульные тесты Google Test
- анализ покрытия кода (gcov/lcov)
- генерация HTML-отчёта покрытия

## Сборка проекта

```bash
make
```
Запуск тестов
```bash
make run
```
Генерация отчёта покрытия
```bash
make coverage
```
После генерации открыть:
```bash
coverage_report/index.html
```
Очистка проекта
```bash
make clean
```
