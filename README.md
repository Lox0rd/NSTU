# Учебный репозиторий

Этот репозиторий создан для систематизации учебных материалов и проектов, выполненных в процессе обучения. Здесь собраны задачи, лабораторные работы, проекты и заметки по различным предметам.

## Структура репозитория

Репозиторий организован по предметам. Каждый предмет имеет свою папку, внутри которой могут быть:
* файлы с кодом (`.py`, `.с`, `.cpp` и т. д.);
* Папки с отчётами и с решениями задач;
* дополнительные материалы (например, PDF‑файлы с лекциями или условиями заданий).
